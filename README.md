# Съезд РОП 2026 — Манифест

Документ согласования съезда руководителей отдела продаж сети «Пикассо Диагностик».

**Площадка:** Казань · Май 2026
**Формат:** 5 дней очно + 12 недель посттренинга
**Авторы программы:** Годнеева В.О., Сергеенко П.В.

## Открыть

После активации GitHub Pages документ откроется по ссылке:
**https://picassokazan-dot.github.io/sezd_rop2026/**

## Включить GitHub Pages

1. Зайти в `Settings` → `Pages`
2. В разделе **Source** выбрать **Deploy from a branch**
3. **Branch:** `main` (или `master`), папка `/` (root)
4. Нажать **Save**
5. Через 1–2 минуты страница станет доступна по ссылке выше

## Структура

- `index.html` — единый файл манифеста (все ассеты встроены, скрины Megaplan Importer в base64)
- `.nojekyll` — отключает обработку через Jekyll (нужно для корректной работы)
- `README.md` — этот файл
